# 📚 Recursos para el Aula

Colección de recursos educativos interactivos para la educación secundaria, desarrollados por **Matias Ara**.

🌐 **Sitio web:** [https://MatiasAra.github.io/recursos-para-el-aula](https://MatiasAra.github.io/recursos-para-el-aula)

---

## ✨ ¿Qué hay acá?

Actividades gamificadas, secuencias interactivas y materiales digitales pensados para el aula. Todos los recursos son de uso libre para fines educativos.

Cada recurso está diseñado con criterio pedagógico: no es "gamificar por gamificar", sino propuestas con objetivos de aprendizaje claros, adaptaciones cuando corresponde, y articulación con el trabajo en papel y carpeta.
